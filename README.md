# Personal Portfolio Website

## 📌 Project Overview
This project is a modern and responsive Personal Portfolio Website developed as Task 3 for the CodeAlpha Frontend Development Internship.

The portfolio showcases personal information, technical skills, featured projects, and contact details in a clean and professional layout. It is designed to provide a strong online presence and demonstrate frontend development skills.

---

## 🚀 Features

- Responsive Design for all screen sizes
- Modern and Attractive User Interface
- Smooth Navigation and Scrolling
- Animated Scroll Reveal Effects
- Skills Showcase Section
- Featured Projects Section
- Contact Form with Interactive Feedback
- Mobile-Friendly Layout
- Single Page Portfolio Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
